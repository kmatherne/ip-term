Simple terminal tool that will quickly output basic information about an IP address.

Accepts both Ipv4 and Ipv6

Example usage and output.

```ipterm 8.8.8.8```


Output

``` 
+--------------+-------------------+
|             IP Info              |
+==============+===================+
| IP           | 8.8.8.8           |
+--------------+-------------------+
| Country      | United States     |
+--------------+-------------------+
| Region       | Virginia          |
+--------------+-------------------+
| City         | Ashburn           |
+--------------+-------------------+
| Zip          | 20149             |
+--------------+-------------------+
| ISP          | Google LLC        |
+--------------+-------------------+
| Organization | Google Public DNS |
+--------------+-------------------+
```
